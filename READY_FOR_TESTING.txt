## 🚀 How to Test

### Step 1: Install Node.js (REQUIRED)
1. Download from: https://nodejs.org (LTS version)
2. Install with default options
3. Restart your command prompt/terminal

### Step 2: Install Dependencies
```bash
cd "C:\Users\yahya\Links\earnify-project\backend"
npm install
```

### Step 3: Start the Server
```bash
npm start
```
You should see: `🚀 Server running on http://localhost:5000`

### Step 4: Test the Application
1. Open `earnify-lab_1.html` in your web browser
2. Test login/logout (should work without errors)
3. Test investment creation:
   - Click any plan → enter amount → "Deposit Proceed"
   - Check Supabase → Table Editor → `investments`
4. Test withdrawal functionality:
   - Click "Withdraw / نکالیں" in sidebar
   - Fill out the withdrawal form
   - Click "Submit Withdrawal Request"
   - Check Supabase → Table Editor → `withdrawals`
